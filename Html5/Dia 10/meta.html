Meta

Defina palavras-chave para mecanismos de pesquisa

meta name="keywords" content="HTML, CSS, JavaScript"

Defina uma descrição da sua página web:

meta name="description" content="Free Web tutorials for HTML and CSS"

Defina o autor de uma página:

meta name="author" content="Igor"

Atualizar documento a cada 30 segundos

meta http-equiv="refresh" content="30"

Definindo o viewport para fazer seu site parecer bom em todos os dispositivos

meta name="viewport" content="width=device-width, initial-scale=1.0"
