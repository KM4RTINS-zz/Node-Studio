Existe três formas de aplicar CSS:

1° Através do atributo Style em uma tag como na tag <h1>.
2° Criando dentro da tag <head> uma tag <style> e aplicando as formatações que a tag irá sofre. Lembrando sempre da sintaxe.
3° Criando um "link:css" na tag <head> que é uma forma de usar o css dentro do html através de um arquivo .css externo.
